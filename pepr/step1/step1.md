### Prerequisites

- Node.js v18.0.0+ (even-numbered releases only)

- npm v10.1.0+

> Node.js v18+ and npm v10.1.0+ are already installed on this environment.

```bash
source ~/.bashrc
```{{exec}}

### Installing Pepr

```bash
npm install -g pepr@0.33.0
```{{exec}}

Let's explore available Pepr commands:

```bash
pepr --help
```{{exec}}
